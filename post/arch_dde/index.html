效果

启动盘的制作

镜像获取：

  1. 官网：https://www.archlinux.org/download/
  2. 镜像站：http://mirrors.ustc.edu.cn/archlinux/iso/

制作启动盘

  1. rufus：https://rufus.akeo.ie/
  2. dd if=镜像路径 of=设备

系统的安装

  1. 通过 bios 或者 boot menu 从U盘启动（uefi+gpt）

  2. 控制台键盘布局默认为 us，跳过不管

  3. 验证启动模式（目录存在即 uefi）

    1

    $　ls /sys/firmware/efi/efivars

  4. ping bilibili.com 测试网络

  5. 更新系统时间：

    1

    $ timedatectl set-ntp true

  6. 硬盘分区 fdisk cgdisk parted mkfs 等命令的使用

    1
2
3
4
5

    为 /boot 建立 200M （随意不要太小就行）分区，mkfs.fat -F 32 格式化分区

为 /、/home、/usr 等建立分区

交换分区的创建 mkswap /dev/sdxx  swapon /dev/sdxx

  7. 挂载分区 mkdir mount 等命令的使用

  8. 修改镜像源 nano、vim 等使用

    1

    位置：/etc/pacman.d/mirrorlist（建议ustc）

  9. 安装基本系统

    1

    # pacstrap /mnt base linux linux-firmware base-devel

配置系统

  1. genfstab -U /mnt >> /mnt/etc/fstab 然后 cat 一下看看有没有问题

  2. arch-chroot /mnt 进入系统

  3. 时区

    1

    $ ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

  4. 本地化 nano /etc/locale.gen

    1
2
3

    en_US.UTF-8 UTF-8          
zh_CN.UTF-8 UTF-8          
zh_TW.UTF-8 UTF-8
    • 接着执行 locale-gen 以生成 locale 讯息
    • 编辑 /etc/locale.conf 写入一下内容：
    1

    LANG=en_US.UTF-8

  5. 主机名
    1 vim /etc/hostname

    1

    myhostname

    2 vim /etc/hosts

    1
2
3

    127.0.0.1   localhost 
::1　　　　　localhost 
127.0.1.1   myhostname.localdomain　　myhostname

  6. 引导程序 grub

    1
2
3

    $ pacman -S grub  efibootmgr       
$ grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=grub         
$ grub-mkconfig -o /boot/grub/grub.cfg

    注：多系统可以安装os-prober，然后将其他系统所在的efi分区挂载，再运行grub-mkconfig

  7. passwd root 为root设置密码

  8. 创建用户

    1
2
3
4

    $　useradd -m -g users -G wheel -s /bin/zsh archie         
$　passwd archie         
$　pacman -S sudo         
$　vim /etc/sudoers

  9. 网络配置

    • 有线连接 systemctl enable dhcpcd 利用 NetworkManager 管理网络 可不设置该项
    • 无线连接 pacman -S iw wpa_supplicant dialog

  10. pacman -S intel-ucode || pacman -S amd-ucode 安装cpu微码

  11. 重启 exit 退出chroot环境 umount -R /mnt 卸载分区 reboot

安装桌面环境

  1. 显卡驱动 pacman -S 驱动包

    • 通用 xf86-video-vesa
    • intel xf86-video-intel
    • 详细参阅 archwiki，fuck nvidia!!!

  2. pacman -S xorg

  3. pacman -S noto-fonts-cjk 中文字体

  4. dde 桌面和一些组件

    1
2

    $ pacman -S deepin      
$ pacman -S deepin-extra  networkmanager

  5. 启用桌面和网络设置
    要使用 DDE 默认的 lightdm greeter，你必须修改 [Seat:*] 部分下的配置文件以声明：

    1
2
3
4

    /etc/lightdm/lightdm.conf
[Seat:*]
...
greeter-session=lightdm-deepin-greeter

    需要注意的是，非 root 用户需要存在有效的主目录才能使 greeter 工作。

    1
2

    $ systemctl enable lightdm
$ systemctl enable NetworkManager

  6. reboot

安装后

  1. 添加 archlinuxcn 源

    1
2
3

    vim /etc/pacman.conf         
[archlinuxcn]          
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch
    • 然后请安装 archlinuxcn-keyring 包以导入 GPG key。
    • 出现问题访问这里：https://www.archlinuxcn.org/gnupg-2-1-and-the-pacman-keyring/

  2. 呵呵一笑

    1
2
3

    $ sudo pacman -S v2ray or yaourt shadowsocksr-libev         
$ pacman -S proxychains-ng        
$ vim /etc/proxychains.conf（记不太清了，请Tab）

  3. 浏览器

    1
2

    $ sudo pacman -S google-chrome         
$ sudo pacman -S firefox firefox-xxxxx(中文包)

  4. oh-my-zsh

    1
2
3
4
5

    $ sudo pacman -S zsh git curl        
$ sudo vim /etc/passwd 
$ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)" 
  or
$ sh -c "$(wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

  5. 中文输入法

    1
2
3
4
5
6

    $ vim ~/.xprofile                 
    export GTK_IM_MODULE=fcitx                 
    export QT_IM_MODULE=fcitx                 
    export XMODIFIERS="@im=fcitx" 
          
$ pacman -S fcitx-im fcitx-configtool

  6. 图标包

    1
2

    $ sudo pacman numix(自行Tab)         
$ yaourt paper icon

  7. grub 美化

    1

    $ sudo pacman -S grub-customizer(自行 Tab，天天 Tab 东西记不住啊)
    • https://www.gnome-look.org/p/1220920

  8. 与 win10 组双系统，arch 使用本地时间

    1

    $ timedatectl set-local-rtc true

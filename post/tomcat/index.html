是什么

The Apache Tomcat® software is an open source implementation of the Java Servlet, JavaServer Pages, Java Expression Language and Java WebSocket technologies. The Java Servlet, JavaServer Pages, Java Expression Language and Java WebSocket specifications are developed under the Java Community Process.

Dynamic web module vesion

Dynamic Web Module version correlates with Servlet API version
关联配置文件:

  1. 普通dynamic web项目只需修改
    web-app/WEB-INF/web.xml: 头约束文件
  2. Maven项目还需修改(javaEE等组件 tomcat提供, Maven项目须得手动添加依赖)
    pom.xml: 保持maven-war-plugin, maven-compiler-plugin and javax.servlet-api 最新版本
